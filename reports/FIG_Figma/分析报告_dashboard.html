框架 1: 护城河判断

核心问题: Figma 的可持续竞争优势是在加深还是侵蚀？

判断: 加深（部分是 — 核心护城河加深，但 AI 范式威胁构成长期侵蚀风险）

证据链:
1. 网络效应自我强化: NDR 从 FY2024 ~130% 升至 Q1 2026 139%，表明平台粘性在增强而非减弱。团队中一人使用 Figma → 全团队必须使用 → 跨部门扩散（ARR>$100K 客户 YoY+46%）[10-Q: mda] [10-K: financial_statements]
2. 技术壁垒持续投入: 浏览器原生 WebGL/WebAssembly 实时协作引擎为核心技术壁垒，2025 年新增 4 个产品（Make/Sites/Buzz/Draw）+ MCP Server 将平台从设计工具扩展为产品开发基础设施，转换成本进一步提高 [10-Q: mda]
3. 反面证据 — AI 范式威胁: Bolt/v0/Lovable 等 AI 代码生成工具可能跳过"先设计再开发"环节，从根本上缩小设计工具 TAM。Figma 通过 Make（AI 原型）和 MCP Server 主动应对，但效果尚需验证 [10-K: risk_factors]
4. Adobe XD 实质放弃，验证 Figma 护城河: Adobe 以 $200亿试图收购 Figma 失败后放弃 XD，间接证明 Figma 的护城河已超过巨头从零复制的经济性 [phase1-data §1]

S 曲线定位: 早中期 — 营收 $10 亿+、增速 40%+，已过早期验证但远未触及 TAM 天花板。PS 估值为主导，尚不适用 PE 估值。

护城河类型: 动态型复合护城河（网络效应 + 转换成本 + 技术壁垒 + 生态系统），具备自我强化特性。

逻辑蕴含:
